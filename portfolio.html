The A-List (App)

The A-List is an application conceived from my experiences as a musician. The biggest issue facing bands and musicians is the sheer difficulty of finding the right people. The A-List aims to allievate that problem with a profile-based musician's network. The user will first create a profile, inputting the various required information (name, age, location, gender, instrument, musical style, etc.) and that profile will be stored in a mongo database. The user can then search for musicians based upon instrument, style, location and gender resulting in a list of matching profiles with contact information. The app utlizes mongoose, node and express to handle the data and back-end operations while the front-end UI utlizes react with a component-based architecture.
